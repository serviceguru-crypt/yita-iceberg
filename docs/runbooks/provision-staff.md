# Provision Staff

1. Sign in as admin or super-admin.
2. Confirm the staff member's role and branch assignment.
3. Use the user provisioning callable/UI when available.
4. Operational roles must have at least one branch.
5. Admin roles should not have branch assignments unless needed for operational testing.
6. Confirm custom claims and `users/{uid}` are in sync.
7. Ask the user to sign out and back in after role changes.

Never let staff self-edit role or branch assignment.
