# Create Branch

1. Sign in as admin or super-admin.
2. Create the branch document through the approved admin process or Firebase Console until a branch UI exists.
3. Required fields:

```text
name
code
isActive=true
settings.orderExpiryMinutes
settings.allowCreditSales
settings.allowSplitPayments
```

4. Assign staff to the branch.
5. Add branch products and opening stock through inventory workflows.
6. Verify branch users can read only their assigned branch.
